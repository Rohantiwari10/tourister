## üì¶ Installation

To run Tourister locally:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/Rohantiwari10/tourister.git
   cd tourister
   ```

2. **Open the project**:

   * If plain HTML/CSS/JS: Open `index.html` in your browser.
   * If using a frontend framework like React:

     ```bash
     npm install
     npm start
     ```
